- 1.如果只选一样东西，面对困局时，毛爷爷最底层的思维习惯是什么？她把眼镜往上推了推，然后说了一句让我停了很久的话。她说，他面对困局从来不是“有没有办法”。他的第一个问题永远是--“现在到底是什么情况”。他花在弄清问题上的时间，永远比别人花在解决问题上的时间多。她说，你看他任何一个阶段的文章，最先出现的一定不是结论，不是口号，不是命令。是调查，是数据，是“谁”，“在哪里”，“有多少”，“为什么”。别人在困局里焦虑的时候，他在做一件看起来最慢的事--把困局拆开，拆到最小单位，然后看每一个最小单位上有没有缝隙。而绝大多数人面对困局的第一反应恰恰相反--他们急于找一个“答案”，在还没看清问题是什么之前就扑上去。
- 2.同时期的其他一些领导者，试图四面出击--前线要打、后方要稳、经济要搞、意识形态要纯。结果每一面都没打透。而毛的逻辑是，在一个时间段里，只有一个矛盾是“主要的”。你把它解决了，其他矛盾会自动降级。
- 3.困局之所以是困局，往往不是因为问题多--是因为你把这十个问题都标成了“紧急”。十个紧急等于没有紧急。十个优先级等于没有优先级。而真正的高手只做一件事--在十个问题里找到那个“源头问题”。源头一解开，下游的九个自己松了。系统论里有一个“杠杆点”原理--在任何复杂系统中，总有少数几个点，你只要用力推动它，整个系统就会发生非线性的改变。找到这个点，四两拨千斤。找不到这个点，千斤也拨不动四两。而毛面对困局时的核心能力，就是在极短的时间内找到当前阶段的那个杠杆点。
- 4.她说，你回去做一个练习。把你现在最头疼的事写下来。然后问自己--这件事里面，哪一个环节是“源头”？哪个问题一旦解决，其他问题就不用解决了？写下来，然后把所有的注意力和资源只放在这一个环节上。其他一切--忍住不动。忍住三天、一周、半个月。你会发现，大部份你当初觉得“必须马上处理”的事，其实不急。急是你自己给的定义。大多数人以为解决问题要全面出击，其实全面出击的潜台词是全面平庸--你把力气匀给了十条线，每条线都只够擦破一层皮。大多数人以为高手比别人多线程，其实真正的高手只有一个线程--他在上一个时间段里只打一个点，打透了再换下一个。困局的第二层裂缝，十个问题里，只有一个是真的。找到它，剩下的自己会排队。

- 23.写作手法：
    - 1.记叙，交待时间，地点，人物，事件起因-经过-结果，用于记事、传奇、小说。
    - 2.描写，精细刻画形象、场景。人物描写：外貌、语言、动作、心理、神态；环境描写：自然环境、社会环境；细分技法：用细节描写、白描、工笔。
    - 3.抒情，直抒胸臆：直接说出喜怒哀乐；间接抒情：借景抒情、托物抒情、借事抒情。
    - 4.议论，亮出观点、讲道理、评判是非，议论文核心。
    - 5.说明，客观介绍外形、功能、成因、用法，说明文专用。

- 28.阶段一，打底--5种表达方式（7天搞定，每天1种），目标，能单独用一种方式完整写一小段，不混杂、基本功扎实。
    - 1.记叙训练。任务，任选一件小事（买菜、散步、一次犯错），只用记叙，写150字，只讲清楚六要素，不加描写抒情议论。自检，有没有多余修饰，事件脉络是否清晰。
    - 2.描写训练。任务，同一个小事，只做描写。比如写路边一棵树，分开练，外形（白描）、风吹晃动（动态）、光影（环境），每段80字。拆分小练习，同样烦心事，借窗外景色带出心情，80字。
    - 3.抒情训练。直接抒情，写一句烦心事，直白说出心里感受，50字；间接抒情，同样烦心事，借窗外景色带出心情，80字。
    - 4.议论训练。找一个小观点（如坚持很重要、手机利弊），写120字简单论述，有观点+简单理由。
    - 5.说明训练。介绍手边物品（水杯、台灯、笔），客观写外形、材质、用途、100字，不带情绪。
    - 6.混合搭配练习。记叙+少量描写，写200字小故事。
    - 7.综合复盘。写300字短文，自然融合五种表达方式。
- 29.阶段二，句子美化--14种修辞（14天，一天吃透1个）
    - 1.每日固定训练模板。先背定义+1个经典例句；基础仿写，拿普通大白话句子，改成相应修辞；扩写，用修辞扩写成50字小片段。
    - 2.每日清单。
- 29.拔高文采--14种表现手法（14天，一天一练）
    - 1.理解手法区别（比如铺垫不等于伏笔，衬托不等于对比）
    - 2.构思场景（花草、人物、往事、风景任选）
    - 3.成文一段，标注哪里用了目标手法
- 30.框架搭建--叙事顺序+篇章结构
    - 1.顺序、倒叙。同一件事，分别用这两种顺序写200字。
    - 2.插叙、补叙。主线故事中间插入回忆，文末补充细节。
    - 3.开门见山VS卒章显志，同一主题，两种开头结尾写法对比。
    - 4.总分/分总/总分总，三段结构各写一小段议论文。
    - 5.层层推进。情感、道理一层比一层深入。
    - 6.双线行文。比如明线养花过程，暗线自己成长心态。
    - 7.留白结尾。写故事，结尾不点透，留回味空间。
- 31.融合贯通综合练习（长期巩固，熟练固化）
    - 1.短篇练（400-600字），每次写一篇，强制用上至少3种修辞+2种表现手法。
    - 2.中篇练（800字标准作文篇幅），完整把控结构、表达方式、所有技巧自然融合。
    - 3.复盘改错。写完逐句标注用了什么手法，哪里运用生硬，修改打磨。
    - 4.仿写范文。找优质散文，记叙文，拆解人家用的手法，复刻仿写。
