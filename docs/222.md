# 关注，非虚构写作，说不定这是你的赛道。（2026年6月14日） 

- 1.首先，心态不要焦躁，网文写作只不过是一个填空题，而你，我的朋友，你不知道如何填而已。下面教你具体怎么下笔填空。
- 2.先快速设计一个最简单的开场小故事。
    - 1.一个开篇故事，4-5个小情节，一句话概括。
    - 2.开场（起）：主角在什么地方，正在做什么？
    - 3.开始（起）：谁出现了，和主角发生了什么互动？
    - 4.进行（承）：事情按照正常的节奏发展？
    - 5.转折（转）：发生了什么意外，打破了正常的节奏？
    - 6.完结（合）：主角用符号自己人设的方式解决了问题，留下一个钩子？
- 3.要学会用电子表格来辅助写作（这也是一种工具啊，要找到更多的工具来帮助自己写作，什么工具都可以，只要能帮助到自己写作。）
- 4.网文可以用这一个公式，重复几十万甚至几百万字。
    - 1.发生了一件事，被A看到、听到或者感觉到。
    - 2.A内心有什么波动？比如吓一跳？生气了？高兴了？
    - 3.A做了什么和说了什么？比如掏手机？拔刀？冷笑？打人？
    - 4.另外一个人B看到，也做反应，有动作有语言。比如慌了，跪了，懵了，笑了，哭了。
    - 5.1-4视为发生了一件事，又从2开始循环。
- 5.举个例子。
    - 1.小弟冲进来，“老大，某某带人把场子围了！”（刺激）
    - 2.主角心里一紧，手心冒汗（主角感觉）
    - 3.不过脸上没什么表情，拉开抽屉扔给他一把枪。（主角动作）
    - 4.“这点事慌什么？”（主角语言）
    - 5.小弟见老大这么稳，也镇定下来，检查枪和子弹。（配角动作）
    - 6.前面这些视作刺激，主角继续
    - 7.配角继续
    - 8.。。。。。。
    - 9.就这样，简单，反复，循环。
    - 10.当然，每一步都可以拆分，细化。
    - 11.少写静态，多写动态。
    - 12.多用动词，少用形容词副词。
    - 13.写难过，就写看见爹妈摆摊，被城管赶。
    - 14.写感动，就写发现女友馒头舍不得吃，留给主角。
    - 15.要信任读者，读者看得懂动作。
    - 16.读者能体会到人物的情绪，你就写对了。
    - 17.人物的情绪变化合理，你就写好了。
- 6.一个故事结构如下：
    | 序列 | 谁？状态 | 情绪 |
    | ---- | -------- | ---- |
    | 1开场 | 很惨，日常苦逼 | 低 |
    | 2切入 |更惨了，被羞辱，被打，被退婚 | 更低 |
    | 3激励（重要）| 忍不了了！想要翻身！| 开始高 |
    | 4进展 |努力，失败|先高后低 |
    | 5转折（重要）| 终于爽了一下，赢了！| 高 |
    | 6二次转折 | 爽过头，出大事了 | 先高后低 |
    | 7危机（重要）| 完蛋了，被打回原形 | 最低 |
    | 8倒入 |顿悟了，获得新力量 | 开始反弹 |
    | 9高潮（重要）| 干翻反派，解决一切 | 最高 |
    | 10结局 | 收获，但埋个新钩子 | 平稳 |
- 7.就我个人经验来说，相较于写小说，其实非虚构写作反而是一个更值得尝试的赛道。
- 8.的确，写小说存在着写出一个爆款，或者一个IP卖出去就财富自由的上限。但是非虚构写作可以提供的是，长期稳定的收益。而根据国内的环境来说，最适合的非虚构写作类型，是科普方向的写作。这个都不用问为什么，社会评论或者其他评论那些的，就国内来说，限制太多，科普反而是更能自由发挥的赛道。
