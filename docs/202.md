# 做了些什么。（2026年3月9日） 

- 1.非专业作家的大量写作指南--有害的前提条件（以及如何对抗它们），詹姆斯。霍顿
- 2.先简单介绍一下背景，我2019年才开始写作。在此之前，我一直处于写作瓶颈期，无法持续稳定地写作超过一个月。然后，有一天，我突然就突破了瓶颈。当然，这背后还有很多故事，而我只是想抓住其中的精髓，并于大家分享。我把它发布在这里，希望能对Substack社区有所帮助。你是否也苦于写作？这些想法或许无法解决你的问题，但请认真思考一下。你是否经常写作，但今天却突然遇到了瓶颈？同样，这些建议或许不能完全帮你突破瓶颈，但我希望你能从中获得一些启发。
- 3.你遇到困难了吗？如何从梦想写作变成真正写作？最简单的答案当然是“写就对了”。但这并不能令人满意。这就象说戒烟的办法就是“戒烟”一样。它忽略了隐藏在自律背后的那场隐秘的心理战争。说“写就对了”就像说“哈哈，赢就行了。”
- 4.几十年来，我一直苦于写作。学生时代，为了赶上截稿日期，避免成绩被扣掉，我拼命写作，但始终无法养成自律的写作习惯。于是，我收集了很多笔本，随身携带一本，里面写满了零碎的句子，待办事项和涂鸦--我同时启动了至少六个写作项目，但没有一个能坚持两个月。我做了很多梦。
- 5.2019年，我改变了。我开始每天写作，这彻底颠覆了我的生活。我的笔记本上满是各种想法，我把它们转化成文章。我练就了一项不可思议的本领：提前两周就能完成学期论文。写作的动力驱使我完成了学术论文的草稿、讲稿，以及一百多篇文章，其中大部份是用钢笔写成的。我很高兴能这么说。我很少有机会能这样陈述我取得的成就。我为此感到自豪，不是处于自恋，而是出于一种欣喜若狂的心情，看着自己亲手打造出曾经看似不可能的东西。这篇文章酝酿已久，因为我想和大家分享我是如何开始做到这一点的。我希望它也能对大家有所帮助。
- 6.问题：有害的前提条件
    首先让我们观察一下：你已经是一位多产的作家了。
    我曾经尝试研究圣雄甘地的著作。但在发现甘地的著作多达98卷，每卷超过400页之后，我不得不重新考虑我的研究计划。
    这听起来似乎很惊人，但如果你审视一下自己的通信记录--每一封邮件、每一条短信、每一篇帖子，以及你发布到推特这个冷冰冰的虚空中的每一条推文--你很可能会发现，到目前为止，你写的东西比甘地一生所写的东西还要多。你是一个互联网用户，一个网民；你每天随口写下的文字，比1980年以前大多数人刻意写的东西还有多。
    然而，不知何故，对你来说真正重要的事情--你一直梦想创作的小说，你想撰写的家族史--却截然不同。它们与你天生的创造力格格不入。为什么呢？
    答案是焦虑，但别着急，我不会跟你讲那些关于你需要更勇敢的玄乎其玄的废话。我更喜欢讨论具体机制，所以下面就来谈谈机制。
- 7.焦虑会以两种方式干扰写作
    - 1.复杂的写作本身就会引发焦虑。心理学认为，焦虑的主要原因之一是目标冲突。目标冲突会造成不适感。而写作充满了目标冲突。写作时，你需要同时进行计划、研究、构思和编辑。这些任务相互冲突，导致持续不断的摩擦和棘手问题。每一次冲突都会消耗精力，都会让人产生想要放弃的念头，迫使你不得不努力才能继续工作。
        担心文字的读者反应也会引发焦虑，尤其是在读者很重要的情况下。试想一下，给同事写一封邮件和给老板写一封邮件--前者是随口而出的，后者则是费尽心思才写出来的。当结果至关重要时，写作就会变得更加困难。
        两种焦虑混杂在一起，会给写作过程增加一层额外的印象管理--这层印象管理几乎不可能完成，因为你管理的不是人的印象，而是你脑海中能召唤出的最残酷的幻影。即使你能坚持下去，写作也会变成一项痛苦不堪、阻力重重的活动。
        摩擦会导致更多问题。一旦写作变成一项高摩擦活动，我们就会开始权衡写作的艰辛与收益。这样思考写作就会将其扭曲成一种不可持续的东西。我们会形成一系列有害的前提条件--关于写作必须具备哪些特质才能“值得”的错误观念。我们将重点讨论其中三点。你可以把它们看作是非写作者对写作的前提条件。
        第一点，写作应当有价值。鉴于写作是一项高成本活动，我们决定只将其用于高回报的项目。
        第二点，写作应该只做一次。因为写作本身就是一种挑战，所以做任何与写作项目无关的事情价值都会降低。
        第三点，写作就应该认真对待。而且，既然写作本身就是一种折磨，那就没必要反复修改润色。一次就够了。谁会傻到去做第二次苦差事呢？
        我所见过的大多数写作问题都源于这些前提条件。的确有一些方法可以让写作变得轻松许多，但如果你认定你的所有写作都必须意义重大、一次完成、而且必须完美无暇，那么在你开始写作之前，所有其他有用的方法都将被排除在外。
        大多数人受困于这些前提条件，不去寻找可持续的写作方法，而是专注于各种激励策略来克服阻力。他们拖延，直到对截止日期的恐惧驱使他们完成项目。他们追逐各种自助建议，渴望获得一种令人振奋的快感，并借此激励自己写作，直到快感消退，不得不寻找下一个目标。或者，他们等待时机，直到环境允许他们重拾决心，但这往往难以实现。然而，他们始终找不到长久的解决之道。
        消除有害的前提条件。
        首先，有个坏消息，你（很可能）永远无法通过自我激励--无论是利用恐惧，兴奋还是自律--来克服写作中的阻力，从而养成可持续的写作习惯。动力会消退，一旦动力消退，阻力就会再次出现，你就会放弃。
        接下来，是个好消息，更有希望的前进方向是找到减少摩擦的方法。最直接的方法是克服你那些有害的先入之见。这并非通过心理治疗就能实现的。相反，你需要采取与这些有害先入之见相反的行动：开始一种与这些先入只见直接相悖的新做法，并坚持下去，直到你的旧信念在新证据面前土崩瓦解。
        我们来谈谈这在实践中是如何体现的。
        消除有害前提1（写作应该有价值）。克服这一前提条件的关键在于重新调整你认为有价值的东西。举个例子，自2019年以来，我发表了超过40万字的文章（其中包括发表在Medium和Substack等博客平台的20万字）。但这仅仅是我写作总量的三分之一左右。其余的都无人问津。为什么？因为它们还无价值。杂乱无章并非指糟糕的文章，而是指我的大部份文字并非作为，而是零散的想法、写给自己的笔记，或是对我兴趣的探索。我的原则是倾听内心的声音--如果它想连续两周只写待办项清单，我就听它的。为什么？嗯，植物生长需要土壤。我敬佩的一位农民曾写到，他90%的工作都是管理土壤。土壤管理的当，植物自然生长。写作也是如此：创造一片充满探索、乐趣和荒诞的沃土，其他的一切都会随之而来。《呼啸山庄》--英语文学史上最伟大的小说之一--就源于勃朗特三姐妹玩维多利亚时代桌面角色扮演游戏的灵感。
        垃圾管理究竟是什么样的？最简单的说法是，写任何你觉得有趣或有用的东西，无论它是否具有公共价值--尤其当它与你最终想要创作的写作类型相关时。以下是我自己写作的两个例子。元写作。这指的是任何能将我的想法外化，以便我梳理思绪的写作。我至少一半的写作都是元写作。它看似平淡无奇，却又令人耳目一新；我写下目标、计划、任务、观察和白日梦。当我的大脑过于嘈杂时，我写作是为了清除杂音；当我的大脑过于沉寂时，我写作是为了创造声音。它太过平淡，不足以称之为日记。但它对我至关重要。元写作帮助我思考。当我的大脑一片混乱时（这种情况经常发生），它帮助我蹒跚前行。我发现它在构建想法、探索内心世界以及规划每日待办事项方面都极其宝贵。元写作还能积蓄动力。在糟糕的日子里，元写作让我保持自律，给我一个提笔写作的理由。因此，我的私人元写作或许是我进行公开写作的最大助力。我的日记里满是这样的页面，开头记者我当天的饮食计划，结尾却冒出一个新想法，或者写出一个文章提纲。你会惊讶地发现，很多时候，那些看似“毫无价值”的文字，会在写到一半的时候，突然变成有价值的东西。缪斯女神会眷顾那些为她腾出空间的人。
        实践。这指的是你的知识库--你写作时所汲取的素材。大多数人的实践积累都是漫无目的的。他们的素材主要储存在脑子里，只有在撰写学期论文时才会添加新的内容。然而，如果你致力于定期写作，那么最令人愉悦的写作活动之一就是有意识地积累实践，通过研究和写作你感兴趣的内容。
        写作时为了学习。我目前在日记里用了几页篇幅记录冰的十九个阶段，还有几页专门记录早期殖民时期美国的伪造活动。诸如此类的信息会定期被我重新利用到文章和其他作品中，而且速度惊人。积累实践，新的想法就会在你写作的过程中自然而然地涌现。对于写作者来说，一个很好的入门方法是从观察和记录的习惯开始，暂时不要考虑分享。一旦你积累了一定的写作动力，就可以开始创作。
