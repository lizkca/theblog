# 央视网，是个巨大的宝藏。（2026年6月29日） 

- 1.番茄这个平台，和传统网文平台不一样。它是免费阅读+广告分成的模式，读者看你的小说不花钱，但平台靠广告赚钱，再分给你。
- 2.这意味着什么？意味着读者没有任何沉没成本。点进来，不喜欢，一秒钟划走。你的书能不能留住人，不能靠签约、不靠推荐位，全靠故事本身。
- 3.也意味着你的收入直接和“读者停留时长”挂钩--他看得越久，广告展示越多，你赚得越多。
- 4.所以在番茄写小说，“写得好”不是最重要的，“让人停不下来”才是。
- 5.那怎么才能让人停不下来？你得懂人性。
- 6.我直接告诉你，番茄上那些月入五万、十万的书，都在死磕这几个方向。每个方向背后，都对应着一种“人性的刚需”。
    - 1.方向一：逆袭--满足“被低估者”的复仇幻想。人性里有一种最原始的快感，叫“我翻身了给你看”。被老板骂了不敢顶嘴，被同事抢功只能忍，被亲戚看不起还要陪笑脸--现实生活里，大部份人都在压抑。他们心里憋着一股劲，总有一天，我要让你们高攀不起。逆袭类小说，就是这股情绪的泄洪口。番茄上跑出来的书，十本里有五六本都是这个路子，赘婿逆袭、废柴逆袭、被退婚之后逆袭、穿越后逆袭、重生后逆袭。核心公式特别简单：主角先被踩到泥里，然后一步步爬上来，最后把曾经看不起他的人统统打脸这里有个细节你得死磕，前期的“压抑”要足够狠。读者要看到主角被欺负得有多惨，后期打脸才有多爽。你要是开头就心软，读者就爽不起来。人性底层逻辑：每个人都渴望被看见、被认可。当现实中得不到时，他们就到小说里去找代餐。
    - 2.方向二：升级--满足“掌控欲”的贪欲。人为什么爱玩游戏？因为游戏里有明确的进度条--你砍一个怪，经验值涨一点；升一级，技能强一分。这种“看得见的成长”，让人上瘾。番茄上跑得好的小说，本质上就是“文字版的游戏”。主角从一个菜鸟开始，练功、突破、打怪、再突破。。。每一个阶段都有明确的目标和奖励。读者跟着主角一起升级，会产生一种“我在变强”的错觉。这种满足感，在现实生活中很难得到的。现实里你努力三个月，可能什么变化都看不到。所以你要做的，就是给读者一个清晰、密集的成长路径。最好每几章就有一次小升级，每几十章有一次大突破。别让他等，等他就走了。人性底层逻辑：人对确定性的回报有天然的渴望。升级系统，就是最快提供这种确定性的方式。
    - 3.方向三：欲望--满足“禁忌”的窥探欲。番茄上，都市文、总裁文、后宫文、年代文里带感情线的，流量一直都很稳。原因很简单--情欲望是人的底层需求，和吃饭喝水一样正常。但你要注意分寸。番茄有审核红线，太露骨不行。真正高明的写法，是“勾着你遐想”而不是“直接给你看”。比如暧昧期的拉扯、误会、吃醋、守护。。。这些情绪上的张力，有时候比直白的描写更抓人。读者会被那种“就差一层窗户纸”的状态吊着，一章一章往下追。尤其是女性向的总裁文、萌宝文，核心从来不是“霸道总裁多有钱”，而是“他只对我一个人好”的独占欲。这种情绪，对很多女性读者来说是致命的吸引力。人性底层逻辑：人对亲密关系和被偏爱的渴望，永远不会过时。
    - 4.方向四：生存--满足“危机感”下的求生欲。这一类包括末世、逃生、系统、无限流等题材。核心就一句话，主角被逼到绝境，靠脑子或运气活下来。为什么好看？因为人的大脑对“危险”天然敏感。你写一个安静的场景，读者可能打呵欠；你写一把刀子架在主角脖子上，三秒钟后就有砍下来，读者的心跳立刻跟着加速。这就是生产类故事的魔力--它激活了人最原始的应激反应。写这类故事，关键就一个：制造真正的绝境，然后给出真正的解法。很多作者写着写着心软了，给主角开金手指开得太明显，读者一看就知道“反正主角死不了”，紧张感瞬间归零。你得让他真的疼。配角可以死，主角可以残，资源可以丢。读者越担心主角，就越放不下书。人性底层逻辑：人在安全的环境里体验危险，会产生一种“劫后余生”的快感。这是过山车的原理，也是生存类小说的原理。
    - 5.方向五：情绪--满足“被理解”的归属感。最后一个方向，可能是最容易被忽视，但后劲最大。有一类小说，情节不那么“爽”，甚至有点虐，但读者追得死去活来。为什么？因为主角的遭遇，戳中了他自己的某段经历。比如原生家庭不幸的主角、被朋友背叛的主角、在底层挣扎看不到希望的主角。。。当读者在故事里看到“原来也有人和我一样”，那种被理解的感觉，比什么逆袭打脸都深刻。这类小说不追求快节奏，追求的是“真实感”和“代入感”。你得写出主角的脆弱、不甘、矛盾，甚至写出他的“不完美”--自私、懦弱、摇摆。越真实，越扎心。越扎心，读者越放不下。人性底层逻辑：每个人都觉得自己是孤独的。能让他找到“同类”的故事，他会当成精神支柱。
- 7.最后给你一个可执行的建议。别五个方向全要。选一个你最擅长的、最想写的，死磕三个月。
