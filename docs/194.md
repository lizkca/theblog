# 工作和幸福是一回事，你知道吗？。（2025年1月13日） 

- 1.我是一个大型外卖应用的开发者，受一项严格的保密协议约束。但是我已经不在乎了，我昨天向公司提交了离职报告。说实话，我希望公司能起诉我，这样一来，这些事情就会曝光。我已经消极工作大约八个月了，只是看着代码被推送到生产环境。一想到自己参与了这台机器，我夜里都睡不着。
- 2.人们总是怀疑算法对用户不利，现实比这更糟。 我是一名后端工程师，每周参加产品会议，产品经理（PM）讨论如何才能挤出额外0.4%的利润，他们把用户当成有待开发的资源。
    - 1.公司有一个“优先配送”服务，你多付2.99美元，就可以更快拿到外卖。这完全是个骗局，根本没有加快派送的速度，而是人为把非优先订单延迟5到10分钟，让你感觉优先订单更快。我们仅仅通过让标准服务变差，就赚取了数百万美元的存利润，而不是真正改善服务。
    - 2.最让我恶心的是“绝望分数”。这是一个隐藏的外送员指标，根据外送员的行为判断他们多想赚钱。如果外送员在晚上10点登陆系统，毫不犹豫地立即接下每一个3美元的垃圾订单，算法会将他们标记为“高度绝望”。一旦被标记，系统就会停止向他们显示高价订单，理由是，既然我们知道他绝望到愿意接受3美元，为什么还要让他看到15美元的订单呢？系统把高价订单留给“休闲”外送员，即那些不愿接低价单的外送员，吸引他们接单，而全职外送员则被碾压成尘埃。
    - 3.公司还会从用户的账单扣除一笔1.50美元的外送员福利费，这个名字让用户感觉在帮助外卖员。实际上，这笔钱流入了，游说反对外送员成立工会的基金，这是公司用于“政策防御”的费用。用户实际上是在为那些高端律师付费，那些律师为削弱外送员的利益而工作。
    - 4.最后，虽然公司不再从外送员的小费里面提成，因为被起诉过，但是却使用其他方法窃取消费。如果算法预测你是“可能支付小费的用户”，而且你很可能会给10美元小费，那么公司只会给外送员可怜的2美元基本派送费。如果你给了0美元小费，公司会给外送员8美元的基本派送费。结果就是，用户的小费并没有奖励外送员，而是在补贴公司。用户给外送员付工资，这样我们就不用付了。
